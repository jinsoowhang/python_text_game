# World Cup Text Game

Text-based soccer World Cup RPG played in the terminal. You're a star player returning from injury for the World Cup Final — make the right plays to lead your country to victory.

## Gameplay

Progress through three stages from midfield to the box, choosing between passing, dribbling, and shooting. Each action has different success odds, creating a risk/reward dynamic.

- Randomized match commentary with typewriter narration effect
- Probability-based outcomes at each decision point
- ASCII art goal celebrations

## Run

```bash
python text_game.py
```
